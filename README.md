# Trabalho-de-Lp2

Primeira etapa do projeto de Lp2 que consiste num novo sistema de gestão academica e de usuarios 
---

##  Objetivo da Primeira Etapa

Implementação Inicial em Java Básico:
  Analisar os documentos da modelagem.
  Estruturar os módulos do sistema - pacotes
  Identificar classes, atributos e métodos das camadas de entidades e serviços.
  Implementar essas estruturas usando Java básico.
  Implementar funcionalidades de teste básico das funcionalidades

---

## Estrutura do Projeto

O projeto está organizado nos seguintes pacotes:

### `Entidades`
Contém as entidades do sistema:
- Usuario
- Discente
- Docente
- Oportunidade
- Inscricao
- Certificado
- Aproveitamento
- Curso
- Grupo
- PPC_Historico

---

### `Services`
Contém as regras de negócio:
- UsuarioService
- OportunidadeService
- InscricaoService
- CertificadoService
- AproveitamentoService
- CursoService
- GrupoService

---

### `Enums`
Enumerações utilizadas:
- Modalidade
- statusAssinatura
- statusGrupo
- TipoOportunidade
- Papel
- StatusInscricao
- StatusOportunidade
- StatusAproveitamento

---

### `main`
Classe principal para testes:
- Main

---

## Funcionalidades Implementadas

- Cadastro e login de usuários
- Mudanca de Curso
- Criação e gerenciamento de oportunidades
- Inscrição de discentes em oportunidades

---

## Testes

Os testes foram realizados por meio da classe `Main`, simulando:

- Criação de usuários
- Criação de oportunidades
- Inscrição em atividades

---

- Java
